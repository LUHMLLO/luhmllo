## Button <small> native </small>

`<button/>` in web development allow users to perform actions or navigate to other pages. They come in various styles to suit different needs and are effective for prompting users to take action to progress within an interface.

<br />

<div data-is="grid" style="--cols: 2; --gap: var(--nm);">
  <div data-is="group" style="gap: var(--sm);">
     Button have simple mouse micro interactions by default

    <div data-is="codeblock">
      <div data-part="example">
        <button>
          <span>button</span>
        </button>
      </div>

      ```html
      <button>
        <span>button</span>
      </button>
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--sm);">
    For outlined buttons you can use the `[data-variant=outline]` attribute

    <div data-is="codeblock">
      <div data-part="example">
        <button data-variant="outline">
          <span>button</span>
        </button>
      </div>

      ```html
      <button data-variant="outline">
        <span>button</span>
      </button>
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--sm);">
    For discrete buttons you can use the `[data-variant=ghost]` attribute

    <div data-is="codeblock">
      <div data-part="example">
        <button data-variant="ghost">
          <span>button</span>
        </button>
      </div>

      ```html
      <button data-variant="ghost">
        <span>button</span>
      </button>
      ```
    </div>

  </div>
</div>

## Extra

<br />

<div data-is="grid" style="--cols: 2; --gap: var(--nm);">
  <div data-is="group" style="gap: var(--sm);">
     Buttons padding can implement optical corrections using `[data-part="lead"]` and `[data-part="trail"]` attributes on its children

    <div data-is="codeblock">
      <div data-part="example">
        <button>
        <i data-is="icon" data-part="lead">asterisk</i>
          <span>button</span>
        </button>
      </div>

      ```html
      <button>
        <span>button</span>
      </button>
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--sm);">
    Buttons with a single child like an `<figure/>`, `<image/>`, `<i/>`, and `<kbd/>` will become circular by defaul `[data-variant=ghost]` attribute

    <div data-is="codeblock">
      <div data-part="example">
        <button>
          <i data-is='icon' >add_2</i>
        </button>
      </div>

      ```html
      <button>
        <i data-is='icon' >add_2</i>
      </button>
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--sm);">
    It dont really has to be an icon, you can use any element, `<figure/>`, `<image/>`, `<div/>`, `<kbd/>`, even another button

    <div data-is="codeblock">
      <div data-part="example">
    			<button data-variant='outline'>
    				<figure
    					data-geometry='circle'
    					data-part='lead'
    					style='background-color: white; height: var(--xl);'>
    					<img src="/images/luisrmelo.png" alt='avatar' decoding='async' loading='eager' />
    				</figure>
    				<span> Luis R. Melo </span>
    				<i data-is='icon' data-part='trail'>expand_all</i>
    			</button>
      </div>

      ```html
      <button data-variant='outline'>
        <figure
          data-geometry='circle'
          data-part='lead'
          style='background-color: white; height: var(--xl);'>
          <img src="/images/luisrmelo.png" alt='avatar' />
        </figure>
        <span> Luis R. Melo </span>
        <i data-is='icon' data-part='trail'>expand_all</i>
      </button>
      ```
    </div>

  </div>
</div>
