<header>
  ## Input

`<input />` elements are the core of form interactivity. They enable users to provide structured data and perform actions across a wide range of use cases. Each input type is optimized for specific formats, aiding both UX and validation at the browser level.

</header>

<div data-is="grid" style="--cols: 2; --gap: var(--space-nm);">
  <div data-is="group" style="gap: var(--space-sm);">
    Standard single-line text input for general text content.

    <div data-is="codeblock">
      <div data-part="example">
        <input type="text" placeholder="Enter text" />
      </div>

      ```html
      <input type="text" placeholder="Enter text" />
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--space-sm);">
    Obscures input for passwords.

    <div data-is="codeblock">
      <div data-part="example">
        <input type="password" placeholder="Enter password" />
      </div>

      ```html
      <input type="password" placeholder="Enter password" />
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--space-sm);">
    Email input validates input pattern natively.

    <div data-is="codeblock">
      <div data-part="example">
        <input type="email" placeholder="name@example.com" />
      </div>

      ```html
      <input type="email" placeholder="name@example.com" />
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--space-sm);">
    Numbers only. Accepts optional min, max, and step attributes.

    <div data-is="codeblock">
      <div data-part="example">
        <input type="number" min="0" max="100" step="1" />
      </div>

      ```html
      <input type="number" min="0" max="100" step="1" />
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--space-sm);">
    Input field for phone numbers. Browser may present a numeric keyboard on mobile.

    <div data-is="codeblock">
      <div data-part="example">
        <input type="tel" placeholder="(555) 555-5555" />
      </div>

      ```html
      <input type="tel" placeholder="(555) 555-5555" />
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--space-sm);">
    Accepts URLs and triggers browser-level validation.

    <div data-is="codeblock">
      <div data-part="example">
        <input type="url" placeholder="https://example.com" />
      </div>

      ```html
      <input type="url" placeholder="https://example.com" />
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--space-sm);">
    File upload input. Accepts files based on type or extension.

    <div data-is="codeblock">
      <div data-part="example">
        <input type="file" accept="image/*" />
      </div>

      ```html
      <input type="file" accept="image/*" />
      ```
    </div>

  </div>

  <div data-is="group" style="gap: var(--space-sm);">

    <div data-is="codeblock">
      <div data-part="example">

      </div>

          <span>Two</span>
        </label>

      <label for="one">

      </label>

      </label>
