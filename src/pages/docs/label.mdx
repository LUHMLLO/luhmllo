## Label <small class="clr-subtext txt-nm txt-lowercase"> native </small>

`<label/>` elements in web development provide a user-friendly way to associate
descriptive text with form controls, such as input elements. By clicking
on a label, users can focus on the corresponding input element, enhancing
accessibility and usability. Labels ensure that forms are easily
understandable and navigable.

<grid data-props="--2" class='outln outln-primary radius-3xs'>
  ```html
  <label>
    label
  </label>
  ```
  <column class='align-center[items] flex place-center p-5xl'>
   <label>label</label>
  </column>
</grid>

### -- parentElement <small class="clr-subtext txt-sm"> PROPS </small>

| prop        | description                                                         | requires |
| ----------- | ------------------------------------------------------------------- | -------- |
| - - field   | changes label behaviour to act as a field wrapper for form elements | -        |

<grid data-props="--2" class="g-sm">
    <column class="grid outln outln-primary radius-3xs">
        ```html
          <label data-props='--field'>
            <span>label</span>
            <input type="text"/>
          </label>
        ```

        <div class="flex place-center p-5xl">
            <label data-props='--field'>
              <span>label</span>
              <input type="text"/>
            </label>
        </div>

    </column>

</grid>

### -- childrenElement <small class="clr-subtext txt-sm"> PROPS </small>

| prop        | description                                                         | requires |
| ----------- | ----------------------------------| -------- |
| - - label   | defines children as field label   | -        |
| - - caption | defines children as field caption | -        |

<grid data-props="--2" class="g-sm">
    <column class="grid outln outln-primary radius-3xs">
        ```html
          <label data-props='--field'>
            <span data-props='--label'>label</span>
            <input type="text"/>
            <span data-props='--caption'>caption</span>
          </label>
        ```

        <div class="flex place-center p-5xl">
          <label data-props='--field'>
            <span data-props='--label'>label</span>
            <input type="text"/>
            <span data-props='--caption'>caption</span>
          </label>
        </div>

    </column>

</grid>
